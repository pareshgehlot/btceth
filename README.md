## Description

An app to get the price of crypto currency

## Installation

git clone --branch master https://github.com/pareshgehlot/btceth
cd btceth
npm install

````

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
````

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
